📂 Excel Files Merger
============================
This Python script merges multiple Excel files from a specified folder into a single Excel file by extracting data from the second sheet of each file.

----------------------------
✨ Features
----------------------------
- Automatically scans a folder for .xlsx files
- Reads data from the second sheet of each Excel file
- Combines all data into one consolidated Excel file
- Skips files with less than two sheets or files that cause errors 

----------------------------
📋 Requirements
----------------------------
- Python 3.x
- pandas
- openpyxl

Install dependencies:

    pip install pandas openpyxl

----------------------------
⚙️ Configuration
----------------------------
- Set the folder_path variable to the folder containing your Excel files.
- Set the output_file variable to the desired output file path where the merged data will be saved.

----------------------------
🕹️ Running the Script
----------------------------
To start the , run: CombineExcels.py
